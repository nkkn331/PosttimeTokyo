# PosttimeTokyo (ポストタイム東京)

<img src="public/logo.png" alt="ロゴ画像" width="300">

## 概要
東京都内にある郵便ポストの「集荷時間」を素早く検索・共有できるウェブアプリケーションです。
公式HPでは網羅されていない詳細な集荷スケジュールを、ユーザー同士の情報投稿によって可視化し、郵送業務の利便性を向上させることを目的としています。

## テーマを選んだ理由
私はよく不用品をフリマアプリで出品します。その際に、購入者が早期到着を希望しており、商品がいつ発送されいつ集荷されるかといった質問を受けることがありました。そんな時にこちらのアプリケーションがあると確認の手間が減り便利であると思い、作成にいたりました。

### きっかけ
フリマアプリでの出品時、購入者から **「いつ発送されますか？」** **「次の集荷に間に合いますか？」** といった問い合わせを頻繁に受けたことがきっかけです。
正確な時間を答えるためにポストまで現認しに行く手間や、外出先で「今ここで出せば一番早く届くポストはどこか」を知りたいシーンが多く、解決策を形にしました。

### 問題点
* **アクセシビリティの欠如**: ポストの集荷時間は現地に行かないと不明な場合が多い。
* **情報の透明性**: 投稿日時を知っているため、情報の鮮度（信頼性）をユーザーが自主的に判断できます。

### 解決策
* **ユーザー投稿型の収集時間シェア**: 任意の投稿の曜日別スケジュールを投稿・閲覧できる仕組み。
* 各投稿の収集情報はユーザーの投稿に基づいており、投稿日時を表示することで、情報鮮度をユーザー自身が判断できる。

## ターゲット層
* フリマアプリを利用する10〜60代の幅広い層
* フリマアプリ以外でも速達などは使わない中でも早く届けたいものがある人

## 主な利用シーン
* フリマアプリなどで集荷時間を質問されたとき
* 郵便物の収集時間を調べたいとき
